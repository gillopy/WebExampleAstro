## ⚙️ Instalación Local

```
git clone https://github.com/gillopy/portfolio.git
cd portfolio
npm install
```

## 🚦 Comandos Útiles

```
npm run dev    # Inicia servidor local en localhost:4321
npm run build  # Genera versión de producción
npm run preview # Previsualiza build local
```
